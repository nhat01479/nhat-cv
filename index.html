Hệ thống quản lý rạp chiếu phim Cinestar

Loại dự án: Dự án nhóm (đầu tháng 09/2023 - cuối tháng 10/2023)

Mô tả dự án:

Dự án mô phỏng lại trang web của Cinestar cho phép người dùng có thêm xem lịch chiếu, đặt vé online hoặc mua vé tại quầy. Ngoài ra cũng cho phép quản lý nhân viên, khách hàng, phim, lịch chiếu, thống kê doanh thu trên toàn hệ thống.

Số lượng thành viên: 4 thành viên

Vai trò bản thân trong dự án: thành viên nhóm

  • Tham gia phân tích nghiệp vụ, nghiên cứu database
  • Trang quản lý: quản lý nhân viên, khách hàng, tài khoản đăng nhập.
  • Trang khách hàng: làm giao diện và API quản lý việc đăng ký, đăng nhập, hiển thị lịch chiếu phim và gửi vé tới email cho khách hàng.
  • Trang thu ngân: giao diện trang thu ngân, trang mua vé và hiển thị lịch chiếu phim.

Công nghệ sử dụng:

  • Làm giao diện bằng ReactJS, Material UI, quản lý state bằng Redux
  • Làm API bằng Java Spring Boot
  • Quản lý mã nguồn bằng Git và GitLab
  • Database: MySQL
  • Upload ảnh bằng Cloudinary
